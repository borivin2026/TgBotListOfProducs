Создай план реализации бота на языке Python для телеграм. 
Бот должен уметь:
1) принимать голосовое сообщение
2) распознавать его с помощью API DeepGram
3) составлять список покупок на освое распознанного сообщения с помощью API Google Gemini
4) отправлять результат в телеграм
5) реагировать на уточнения, если что-то распозналось не так, или если пользователь добавляет уточнения. Т.е. например: хлею - 3 штуки, хлеб - 2 штуки. 
6) в боте должна быть кнопка принять как окончательный список покупок. 
7) в каждом ответе есть приложение в виде csv файла, чтобы можно было ответ скачать

всю историю покупок с пользователем хранить в локальной sqlite базе данных
при этом пользователь может после покупки прикрепить скриншот чека. Бот должен уметь распознавать товары в чеке и сохранять их в отдельной таблице, с датой и т.д. Чеки могут немного отличаться по содержанию, поэтому важно чтобы был некий минимальный унифицированный формат


писать логи работы бота в файл.

бот на локальном ПК тестировать через лонг-поллинг, для развертывания на сервере использовать webhooks

бот будет развертываться на сервере PythonAnywhere
