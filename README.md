# Smart Shopping List Bot

Телеграм-бот для составления списков покупок с помощью голоса, интеллектуальной обработки через Google Gemini и оцифровки чеков.

## 🚀 Быстрый запуск

### 1. Подготовка окружения
Убедитесь, что у вас установлен Python 3.10 или выше.

```bash
# Клонируйте репозиторий (если применимо)
cd TgBotListOfProducs

# Установите зависимости
pip install -r requirements.txt
```

### 2. Настройка ключей
Создайте или откройте файл `.env` в корневой папке и добавьте следующие ключи:

```env
TELEGRAM_TOKEN=ваш_токен_от_BotFather
DEEPGRAM_API_KEY=ваш_ключ_deepgram
GEMINI_API_KEY=ваш_ключ_google_gemini
```

### 3. Запуск тестов (опционально)
Перед запуском бота можно проверить работу API:
```bash
# Проверка Gemini
python scripts/test_gemini.py

# Проверка Deepgram (нужен тестовый аудиофайл .ogg или .mp3)
python scripts/test_deepgram.py path/to/audio.ogg
```

### 4. Запуск бота
```bash
python bot/main.py
```

## 🛠 Основные функции
- **🎙 Голос:** Отправьте голосовое сообщение со списком товаров.
- **🔄 Уточнения:** Добавляйте товары или меняйте их количество просто продолжая диалог.
- **📂 CSV Экспорт:** Бот присылает файл со списком после каждого обновления.
- **📸 Чеки:** Команда `/receipt` позволяет сфотографировать бумажный чек и сохранить его данные в базу.
- **💾 История:** Все списки и чеки сохраняются в локальной базе SQLite (`database/bot_database.db`).
